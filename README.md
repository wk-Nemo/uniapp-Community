# uniapp-music
这是一个uniapp的实战项目！为了更好的学习熟悉uniapp的开发。

## 使用方法
- 进入`frontend`文件夹，使用安装相关依赖
```
npm install
```
- 根据package.json的配置选择要启动的
  - 在H5端运行，不带有云函数功能

  ```
  npm run server
  ```
  - 在微信小程序上运行，带有云函数功能。会生成`dist`文件夹，使用微信开发者工具打开即可。
  ```
  npm run dev:mp-weixin
  ```
